# Changelog

## [0.5.0] - 2025-12-16
- 录制时显示四角闪烁边框提示
- 托盘图标录制状态改用主题色
- 编辑器窗口支持独立工作，不受新录制影响
- 应用改为菜单栏模式，不显示在 Dock 和 Cmd+Tab

## [0.4.0] - 2025-12-16
- 新增 GIF 录后编辑器，支持裁剪、调整分辨率和帧率
- 新增 Filmstrip 时间线，可拖动选择导出范围
- 支持多种循环模式（无限循环/单次/来回）
- 实时预估导出文件大小

## [0.3.3] - 2025-12-15
- 统一选择器窗口为 Warm Academic 设计风格

## [0.3.2] - 2025-12-15
- 将选择器窗口重构为 React 组件

## [0.3.0] - 2025-12-14
- 重构 GIF 保存为后台异步编码
- 全局快捷键改由 Rust 端注册

## [0.2.0] - 2025-12-13
- 初始版本，支持区域截图和 GIF 录制
