# Changelog

## [0.3.2] - 2025-12-16
- 优化 GIF 录制保存流程，界面不再卡顿
- 简化主界面，在选区时选择截图/GIF模式
- 暂时禁用视频录制功能

## [0.3.1] - 2025-12-16
- 修复快捷键触发选区后立即退出的问题
- 快捷键改为 ⇧+⌥+A（Shift+Option+A）
- 移除 Capture 按钮，完全使用快捷键操作

## [0.3.0] - 2025-12-16
- 截图自动复制到剪切板，可直接粘贴使用
- 修复截图区域坐标计算错误
- 修复截图时选区边框被截入的问题

## [0.2.0] - 2025-12-16
- 全新紧凑界面设计，窗口更小巧
- 新增 Image/GIF/Video 格式切换
- 应用温暖学术风格（Warm Academic）设计

## [0.1.0] - 2025-12-16
- 初始版本发布
- 基于 Tauri 2 + React 19 的 GIF 截图桌面应用
