# Changelog

## [0.11.0] - 2025-12-17
- 首页新增历史记录，快速访问过往截图和 GIF
- 托盘菜单新增「Show Lovshot」显示主窗口
- 热更新后自动恢复编辑状态
- 时间轴 hover 即可实时预览帧位置

## [0.10.0] - 2025-12-16
- 新增「排除标题栏」功能，窗口截图可只保留内容区域
- 支持常见应用预设（Chrome/Safari/VS Code 等）

## [0.9.1] - 2025-12-16
- 修复 GIF 重复导出报错的问题

## [0.9.0] - 2025-12-16
- 时间轴支持点击/拖拽预览任意帧
- 编辑器上方实时显示当前帧画面

## [0.8.0] - 2025-12-16
- 选区确认后支持拖拽边界调整大小
- 窗口高亮改为纯预览，不影响正常操作

## [0.7.1] - 2025-12-16
- 修复 Retina 显示器截图区域只有一半的问题

## [0.7.0] - 2025-12-16
- GIF 导出新增画质、速度、FPS 独立调节
- 导出时可选择保存路径
- 速度调节影响输出时长和文件大小

## [0.6.1] - 2025-12-16
- 重构后端代码结构，提升可维护性

## [0.6.0] - 2025-12-16
- 支持检测并选中 Dock 栏
- 优化截图流程，避免截图包含选择框

## [0.5.0] - 2025-12-16
- 录制时显示四角闪烁边框提示
- 托盘图标录制状态改用主题色
- 编辑器窗口支持独立工作，不受新录制影响
- 应用改为菜单栏模式，不显示在 Dock 和 Cmd+Tab

## [0.4.0] - 2025-12-16
- 新增 GIF 录后编辑器，支持裁剪、调整分辨率和帧率
- 新增 Filmstrip 时间线，可拖动选择导出范围
- 支持多种循环模式（无限循环/单次/来回）
- 实时预估导出文件大小

## [0.3.3] - 2025-12-15
- 统一选择器窗口为 Warm Academic 设计风格

## [0.3.2] - 2025-12-15
- 将选择器窗口重构为 React 组件

## [0.3.0] - 2025-12-14
- 重构 GIF 保存为后台异步编码
- 全局快捷键改由 Rust 端注册

## [0.2.0] - 2025-12-13
- 初始版本，支持区域截图和 GIF 录制
